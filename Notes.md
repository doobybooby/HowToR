# Learn How To R
* You have to compile the code one line at a time


## Printing Char/Strings
* You can use "Quotation Marks" or 'apostrofy' to print out a char/string.
* It doesn't matter which one you use
###### EX1
```R
"Hello World!" 
```
returns
```R
> "Hello World!"
[1] "Hello World!"
```
###### EX2
```R
'Bye World!"
```
returns
```R
> 'Bye World!'
[1] 'Bye World!'
```
